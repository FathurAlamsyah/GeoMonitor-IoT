# GeoMonitor-IoT
A smart IoT platform for monitoring environmental and ground conditions, including soil moisture, soil temperature, vibration, and ground movement.

GeoMonitor IoT is a smart monitoring system designed to collect and visualize environmental and ground condition data in real time.

The system measures:
- Ground Vibration
- Soil Moisture
- Soil Temperature
- Surface Displacement / Inclination

Applications include environmental monitoring, landslide observation, agricultural research, and educational projects.
